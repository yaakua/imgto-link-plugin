# FaFaFa-全部发 商店上架文案（中文）

## 插件名称
FaFaFa-全部发

## 简短描述
从 fafafa.ai mdeditor 一键发布到多个内容平台，自动检测登录状态并尽可能保留排版。

## 详细描述
FaFaFa-全部发 是 fafafa.ai 官方浏览器扩展，面向需要把一篇 Markdown 内容同步到多个平台的创作者、运营和技术作者。

安装后，你可以直接在 fafafa.ai 的 mdeditor 中点击“发布”，检测已登录的平台账号，选择目标平台，然后让插件自动打开对应编辑器并填充内容。

核心能力：
- 一次编辑，多平台分发
- 自动检测平台登录状态
- 支持富文本 / Markdown / 微信样式内容桥接
- 尽可能保留排版效果，降低二次整理成本
- 本地浏览器内执行，不依赖插件开发者服务器转发正文内容

适用场景：
- 技术博客多平台同步
- 内容运营批量分发
- 微信公众号 / 小红书 / 掘金 / CSDN 等平台草稿同步

## 权限说明
- `scripting`：向目标平台编辑器注入内容
- `cookies`：检查目标平台登录状态
- `tabGroups` / `activeTab`：组织和管理自动打开的发布标签页
- `debugger`：部分平台需要模拟粘贴行为以保留排版
- `clipboardRead`：读取富文本 HTML 内容用于同步
- `storage`：缓存必要的本地状态
- `offscreen` / `declarativeNetRequest`：支持某些平台的登录检测与兼容流程

## 隐私摘要
- 不收集个人信息
- 不将正文内容上传到插件开发者服务器
- 仅在你主动触发发布时访问目标平台
