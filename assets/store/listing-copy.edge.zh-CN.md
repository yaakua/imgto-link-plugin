# FaFaFa-全部发 Edge 商店上架文案（中文）

## 产品名称
FaFaFa-全部发

## 简短描述
从 fafafa.ai mdeditor 一键发布到多个内容平台，自动检测登录状态并尽可能保留排版。

## 详细描述
FaFaFa-全部发 是 fafafa.ai 官方浏览器扩展，帮助创作者把同一篇 Markdown 内容快速同步到多个内容平台。

安装后，用户可直接在 fafafa.ai 的 mdeditor 中点击“发布”，检测当前浏览器中已登录的平台账号，选择目标平台后由插件自动打开目标编辑器并填充内容。

### 核心能力
- 一次编辑，多平台分发
- 自动检测平台登录状态
- 支持富文本 / Markdown / 微信样式内容桥接
- 尽可能保留原始排版，减少手动整理
- 在本地浏览器中执行，不经由插件开发者服务器转发正文

### 典型场景
- 技术博客多平台分发
- 内容运营批量同步
- 微信公众号 / 小红书 / 掘金 / CSDN 等平台草稿同步

## 审核说明
- 插件仅在用户主动点击“发布”后运行
- 不在后台自动代发
- 正文内容默认仅在本地浏览器内处理
- 站点访问权限仅用于登录检测、打开编辑器、填充内容

## 权限说明
- `scripting`：向目标平台编辑器注入内容
- `cookies`：检查目标平台登录状态
- `tabGroups` / `activeTab`：组织发布过程中打开的标签页
- `debugger`：部分平台需要模拟粘贴以保留排版
- `clipboardRead`：读取富文本 HTML 用于发布流程
- `storage`：保存必要本地状态
- `offscreen` / `declarativeNetRequest`：支持登录检测与兼容流程

## 隐私摘要
- 不收集个人信息
- 不把正文上传到插件开发者服务器
- 仅在用户主动触发发布时访问目标平台
